Co-founded and led engineering for Rover (née Documatic), a code analysis platform for identifying bugs in large-scale codebases. We built a powerful codebase parsing + graphing system which connected disparate pieces of code across codebases and teams, and an AI agent to traverse the codebase to identify tricky logic bugs. We built genuinely fantastic tech, grew some solid ARR, raised over $2m in seed funding, built a team of five.
